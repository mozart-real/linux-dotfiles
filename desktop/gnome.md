# 🧩 Extensões do GNOME

Este arquivo documenta as extensões utilizadas no meu setup para garantir a consistência visual e funcional do desktop, separando o que é específico do sistema e o que é universal.

---

## 🟦 Zorin OS
> **Informações do Sistema:**
> - **SO:** Zorin OS 18.1
> - **Interface:** GNOME Shell 46.0

### Extensões Ativas:

#### 🖥️ Interface & Visual
* **Blur my Shell** (`blur-my-shell@aunetx`): Efeito de desfoque (blur) nos elementos da interface.
* **Desktop Cube** (`desktop-cube@schneegans.github.com`): Efeito 3D de cubo na troca de workspaces.
* **User Themes** (`user-theme@gnome-shell-extensions.gcampax.github.com`): Permite carregar temas de shell customizados.

#### ⚙️ Integração Zorin (Nativas)
* **Zorin Tiling Shell**: Gestão de janelas lado a lado (tiling).
* **Zorin Connect**: Sincronização com smartphone.
* **Zorin AppIndicator**: Suporte para ícones na bandeja do sistema.
* **Zorin Desktop Icons**: Suporte para ícones no ambiente de trabalho.
* **Zorin Printers**: Atalho para gestão de impressoras.

#### 🛠️ Utilitários & Gestos
* **Add to Desktop**: Atalhos rápidos no ambiente de trabalho.
* **Emoji Copy**: Menu de busca e cópia de emojis.
* **X11 Gestures**: Gestos de touchpad para o servidor X11.
* **Drive Menu**: Acesso rápido a discos e pens USB.
