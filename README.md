# 📂 My Dotfiles | Personal System Environment

Repositório centralizado para meus arquivos de configuração. Organizado de forma modular para facilitar a manutenção e a portabilidade entre diferentes sistemas Linux.

---

## 🏗️ Estrutura do Repositório

Diferente de repositórios planos, este projeto separa as responsabilidades por diretório, permitindo que você escolha exatamente o que deseja sincronizar.

### 💻 Terminal
Localizado em `/terminal`
* **Shell:** Configurações de `.zshrc` ou `.bashrc`.
* **Prompt:** Temas customizados (Starship/Oh My Zsh).
* **Aliáses:** Atalhos produtivos para desenvolvimento e busca.

### 🎨 Estética & Interface
Localizado em `/wallpaper` e `/themes`
* **Wallpapers:** Curadoria de papéis de parede em alta definição.
* **Cores:** Paletas consistentes (Catppuccin, Tokyo Night ou Drácula).
* **Fontes:** Configurações para Nerd Fonts e renderização de texto.

### ⚙️ WM & Compositores
Localizado em `/wm`
* **Gerenciadores:** Configurações para Hyprland, i3, ou Zorin/GNOME.
* **Barra:** Arquivos de estilo para Waybar ou Polybar.
* **Menus:** Temas para Rofi ou Wofi.

## 🚀 Como Utilizar

Este repositório foi desenhado para ser usado com **links simbólicos**. Em vez de copiar e colar, você vincula as pastas deste repositório diretamente na sua pasta `~/.config`.

### Exemplo de Organização:
1. Faça o clone: `git clone https://github.com/mozart-real/linux-dotfiles.git`
2. Mova ou vincule as pastas desejadas:
   - Configurações de terminal para `~/.config/terminal`
   - Papéis de parede para sua pasta de imagens ou script de BG.
